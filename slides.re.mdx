# React Native とは

<Row>
  <Column align="left">
    <Text>Reactを使ってネイティブアプリを開発できる</Text>
    <Text>JSで iOS/Android 両方のアプリを作成</Text>
    <Text>WebView とは違い、直接 Native レイヤーを触りに行く</Text>
    <Text>開発元のMetaをはじめ、MS、Amazon、Shopify、Discord等大企業が導入</Text>
  </Column>
</Row>

# WebView
<Row>
  <Column align="left">
    <Text>ReactなどのWebの資産をそのまま使う</Text>
    <Text>OS準拠のブラウザに則って動く</Text>
      <Text>iOS なら JSC, Android なら v8 など</Text>
    <Text>ネイティブアプリ側との連携が挟まるため、スムーズなUXを提供しづらい</Text>
  </Column>
</Row>

# New Architecture

<Row>
  <Column align="left">
    <Text>ネイティブ層との通信を同期的に行う仕組み</Text>
    <Text>以前はJSON経由で全て非同期に行われていた</Text>
    <Text>• 2022 年から段階的に導入</Text>
    <Text>• 2024年でデフォルトで使われるように</Text>
    <Text>• 今年リリースされた0.82からLegacy Architectureの撤廃</Text>
  </Column>
